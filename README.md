# DS-4002-CS3: Forecast-Driven Dynamic Pricing Case Study

## Overview
This case study challenges a 2nd year UVA data science student to investigate whether forecast-driven dynamic pricing can generate more revenue than a flat-rate static pricing system using NYC Yellow Taxi data.

## Repository Structure
- `HOOK/` — One-page hook document to introduce the case study
- `RUBRIC/` — Grading rubric describing deliverables and expectations
- `DATA/` — Data download instructions (dataset too large for GitHub)
- `SCRIPTS/` — Four Jupyter notebooks for the full analysis pipeline
- `OUTPUT/` — Visualizations generated during the analysis
- `REFERENCES/` — Reference list

## Instructions
See `DATA/data_download_instructions.md` to download the NYC TLC dataset, then run the notebooks in `SCRIPTS/` in order (01 → 04).
